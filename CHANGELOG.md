# Список изменений

Все заметные изменения в этом проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Семантического Версионирования](https://semver.org/lang/ru/spec/v2.0.0.html).

## [1.0.2] - 2026-04-15
### Изменено
- `MaxBotClient::editMessage` переведен на `PUT /messages` с передачей `message_id` в query-параметрах.
- Внутренний метод `request` расширен параметром `$extraQuery` для добавления служебных query-параметров.

## [1.0.1] - 2026-03-25
### Добавлено
- Добавлен метод `editMessage` в `MaxBotClient`.

## [1.0.0] - 2026-03-18
### Добавлено
- Начальный выпуск.
- Канал уведомлений Max для Laravel.
- MaxBotClient для взаимодействия с API Max.
- MaxMessage с поддержкой текста, кнопок и ссылок.
